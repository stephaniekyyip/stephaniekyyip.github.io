Smart Light System

Final project in Internet of Things class

Overview

For the final project in our Internet of Things course, my team decided to create a smart light system to remotely control a LED bulb. The idea behind the smart light was to conserve energy by being able to remotely turn off lights and adjust the brightness depending on the ambient light.

How the Smart Light Works

We interfaced the LED bulb, along with a photoresistor and a transistor, with a Raspberry Pi. The photoresistor was used to determine the ambient brightness of the room in order to adjust the LED brightness accordingly. The transistor used with the battery pack allowed us to vary the brightness of the LED using the pulse width modulation (PWM) output from the Raspberry Pi.

My Role within the Project

I was responsible for the software side of the project. In order to control the LED, I wrote Python scripts to adjust the LED brightness by varying the PWM output. I also wrote a Python script to read the input from the photoresistor to allow the option of automatically adjusting the brightness depending on the ambient light in the room. After setting up an Apache server on the Raspberry Pi, I created the user interface using HTML for users to select different LED settings and used PHP to retrieve the user input in order to execute the Python scripts.
